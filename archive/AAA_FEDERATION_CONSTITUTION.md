
## III. Class Hierarchy — APEX / ASI / AGI

### AGI — Agent Intelligence (Worker Layer)
Domain reasoning, planning, execution, tool use. GEOX, WEALTH, ENGINEER occupy this layer because they act on world models and execute tasks.

### ASI — Agent Superintelligence (Validator-Heart Layer)
Higher-order integrity checking, witness alignment, consequence-sensitive evaluation. VALIDATOR-Agent behaves as an ASI by checking FK integrity and verdict/stage validity. Not "superintelligence" in marketing terms — rather, **coherence at scale**.

### APEX — Agent Apex (Sovereign Layer)
Audit, escalation, constitutional override logic, federation-level arbitration. AUDITOR-Agent and ARIF-Perplexity both occupy APEX roles — one internal floor enforcement, one external sovereign reasoning.

### COORD — Coordination Layer (Missing)
The router/dispatcher that owns routing across all AGI/ASI/APEX agents. Currently absent from the roster. This is the **AAA-Agent** gap.
